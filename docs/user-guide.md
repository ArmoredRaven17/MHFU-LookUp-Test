# User Guide

MHFU LookUp is an **offline** reference for Monster Hunter Freedom Unite (MHP2G). Pick a tab on the left —
most have a **search box** at the top that filters as you type. The app also has in-app **Help** and
**About** tabs covering this same material.

## Tabs

| Tab | What it shows |
|---|---|
| **Monsters** | Hitzones (cut / impact / shot + each element, color-coded), stagger & break thresholds, ailment tolerances, and carve / capture / break reward tables per rank. |
| **Weapons** | Upgrade trees per weapon type; sharpness, element/status, slots, affinity; type-specific detail (bow charges & coatings, bowgun ammo, gunlance shelling, hunting-horn songs). "Filter…" narrows the list. |
| **Armor Sets** | Per-piece defense, resistances, slots and skill points; the skills that activate when the set is worn (<span>green</span> = good, <span>red</span> = bad). Blademaster/Gunner and Male/Female toggles. |
| **Armor Skills** | Every armor skill grouped by category, with point thresholds and what each tier does. |
| **Decorations** | Jewels and the skills (with points) they grant, slot cost, and crafting recipes (with material icons). List by name or by skill. |
| **Quests** | Organized by hub and rank; target monster(s)/delivery items, KEY/URGENT badges, objective, fee, reward and unlock conditions. |
| **Gathering** | Per-area node tables; Zone/Node/Type stay fixed on the left while you scroll the rank columns (Low/High/G, Training, Treasure). |
| **Items** | The full item list with descriptions, rarity, carry limit and sell value. **Account Items** also show a Pokke-point value. |
| **Combo List** | Combination recipes (Product = Item 1 + Item 2) with success chance and quantity; grouped into normal, Treasure-Hunt-only and Alchemy-only. |
| **Treasures** | Treasure-quest items with point values and where to find each; a ★ marks items that count toward a Guild Card award. |
| **Kitchen** | Felyne Kitchen recipes (two ingredients → an effect). Pick the number of chefs to see that level's ingredients & recipes; effects are color-coded. A "Felyne Whim Skills" page lists every whim skill. |
| **Trenya** | Items Trenya brings back from sea trips, by destination; choose how many Pokke Points to fund him with (higher tiers = rarer items), grouped by category. |
| **Pokke Farm** | Items obtainable from each farm area (Mining, Fishing, Insect Thicket, Bug Tree, …), listed by upgrade tier; Field Rows also shows plantable & fertilizer items. |
| **Awards** | Every Guild Card award with its icon, description, and how to earn it. |
| **Settings** | UI scale, window/taskbar icon, app color theme, and the monster icon shown on each tab (with "Restore Defaults"). |
| **Help** | This guide, in-app. |
| **About** | Data sources & attribution, trademarks/ownership, and disclaimers. |

## Tips

- **Search** — most list tabs filter as you type (name, and often description or condition).
- **Wide tables** — the mouse wheel always scrolls **vertically**; to move a wide table sideways, drag its
  horizontal scrollbar (it appears only when needed).
- **Everything is offline** — all data is stored locally, so the app works with no internet connection.
- **Make it yours** — Settings lets you change the theme, the per-tab icons, and the UI scale.

> Data is compiled from community wikis and is factual game data; English names follow Capcom's official
> localization. See the **About** tab for full attribution. Errors may remain — verify critical values
> against the original sources.
